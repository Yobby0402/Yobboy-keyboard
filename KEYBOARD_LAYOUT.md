## 🔧 配置说明

### GPIO 配置

```c
// HC165 按键扫描
#define PIN_DATA  GPIO_NUM_XX  // 数据引脚
#define PIN_CLK   GPIO_NUM_XX  // 时钟引脚
#define PIN_SHLD  GPIO_NUM_XX  // 锁存引脚

// LED 控制
#define LIGHTMAP_GPIO 48       // WS2812 数据引脚
#define LIGHTMAP_NUM  71       // LED 总数
```
